Versioning:
major.minor.current
e.g. 0.0.1 = unreleased.none of the milestones.1

Major versions go up every major release (e.g. beta to launch)
Minor versions go up every milestone (e.g. implemented multi-unit selection)
Current versions go up every milestone in the milestone (and go up a lot)
